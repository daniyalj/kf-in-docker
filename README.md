# Kf in a Docker image
=========

Dockerfile for running [Kf](https://cloud.google.com/migrate/kf/docs) binary in a Docker image

Requirements
------------

- Docker

Example build
----------------

```
$ docker build -t gcr.io/myproject/daniyalj/kf-in-docker:v1
```
